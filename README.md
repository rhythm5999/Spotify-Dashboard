# Spotify Dashboard - Top 50 World Songs 🎶

## 📌 Overview
This project analyzes Spotify’s **Top 50 Global Songs dataset** and builds an interactive dashboard to uncover insights about popularity, duration, explicit content, and release trends.

## 📊 Dataset
- Source: Kaggle / Spotify API
- Columns: `date`, `position`, `song`, `artist`, `popularity`, `duration_ms`, `album_type`, `total_tracks`, `release_date`, `is_explicit`, `album_cover_url`

## 🛠️ Tools & Tech
- **Python**: pandas, matplotlib, seaborn
- **SQL**: PostgreSQL, MySQL
- **Power BI**: Dashboard creation
- **Gamma**: Presentation

## 🔍 Steps
1. Load dataset in Python
2. Perform EDA (clean missing values, visualize distributions)
3. Run SQL queries for deeper insights
4. Build Power BI dashboard with KPIs
5. Present findings with Gamma

## 📈 Dashboard Highlights
- Total songs, artists, albums
- Average popularity & duration
- Explicit vs non-explicit ratio
- Release year trends
- Top 10 artists by popularity

## 🚀 Results
- Most popular artists & songs
- Trends in explicit content
- Insights into song duration and album types
